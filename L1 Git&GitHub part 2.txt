Some more commands for git are as follows:-

git branch feature :- create a new branch named feature.
git checkout feature :- This command means that the head is pointing towards feature branch.Which means all new comments you made added towards the head.
git merge feature :- merge branch with main.
git clone url :- downloading Repository from Github to local hardrive.

//From where you have forked the project that url is called upstream url.

What is Pull Request?
